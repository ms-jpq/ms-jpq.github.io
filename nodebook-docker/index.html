Nodebook - Docker in Docker

Managed to put Nodebook inside a Docker image.

All credits go to the original author - netgusto.

To Use

docker run -it --rm \
   --privileged \
   -p 8080:80 \
   -v "$PWD"/docker:/var/lib/docker \
   -v "$PWD"/notebooks:/notebooks \
   msjpq/nodebook

Remove -v "$PWD"/docker:/var/lib/docker for Mac.

Misc

--privileged is required to Docker in Docker to work

"$PWD"/docker:/var/lib/docker allows for caching
